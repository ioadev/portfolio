Smart Brain Web App

SmartBrain is a full-stack web application I built, demonstrating my skills in both front-end and back-end development. The user interface, crafted with React, allows users to submit image URLs and displays detected faces with bounding boxes using the Clarifai API.

For robust security, the backend utilizes Express.js and PostgreSQL to manage user registration, login, and data storage. Bcrypt ensures passwords are securely hashed, never stored in plain text. Additionally, SmartBrain tracks user logins and image submissions, enabling user ranking based on their engagement. This project showcases my ability to build secure, user-friendly, and feature-rich web applications, utilizing various technologies effectively.
